2 Required Application Documents

Exact list from NOFO Part 2. Upload exactly as named and formatted. Missing or incomplete = disqualification risk.

  • ✓
    1. SF-424 Application Form
    PDF · Org legal name, UEI, address, project title, funding request, duration, key contacts. Pull from SAM.gov registration.
    Template available
  • ✓
    2. Technical Volume
    PDF · Max 30 pages. Full project description, technical approach, innovation, feedstock plan, recovery yields/costs, supply-chain impact, market analysis, risk mitigation, job creation, environmental benefits. Core narrative: cobalt/copper/zinc/tungsten/silver recycling from e-waste.
    Core deliverable
  • ✓
    3. Letters of Commitment (1 page each)
    PDF · 1 page per letter. Signed letters from: Texas A&M (R&D scope), feedstock suppliers (Indonesia/Africa + domestic), offtake buyers (battery/defense with minimum tons/year), any subrecipients/partners. Target 3–5 strong letters.
    Biggest gap — secure now
  • ✓
    4. Impacted Indian Tribes Documentation
    PDF. If project affects tribal lands/resources: consultation records. If none apply: statement that none apply.
    Usually N/A statement
  • ✓
    5. Statement of Project Objectives (SOPO)
    MS Word · Max 8 pages. Measurable goals, work plan, milestones, outcomes. 60 months: define (1–12), build (13–30), scale (31–60). 350–450t in 24mo, 1,500–2,500 t/yr by Year 5. Draft exists — customize with final numbers.
    Draft exists — finalize
  • ✓
    6. Budget Justification Workbook
    MS Excel. Line-item budget, cost share breakdown, justification for every cost category. Must show 50/50 split. Every line ties to the SOPO.
    Needs full build
  • ✓
    7. Subrecipient Budget Justification
    MS Excel. Same as above for any partners/subrecipients (e.g., Texas A&M scope and costs). Only required if using subrecipients.
    If using university
  • ✓
    8–11. FFRDC / Foreign Work / Waiver Documents
    PDF. Only if using FFRDC labs or any foreign work/participation. Include waiver requests if applicable.
    Likely N/A
  • ✓
    12. Resumes of Key Personnel
    PDF · Max 3 pages per person. PI, project manager, technical leads. Emphasize experience in metal recovery, refining, hydrometallurgy, project management.
    Pull CVs now
  • ✓
    13. Current & Pending Support
    PDF. For every senior/key person: all other funding sources and % effort commitment. Standard form per person.
    Standard form
  • ✓
    14. Transparency of Foreign Connections
    PDF · Business Sensitive. Any foreign ownership, affiliations, or funding. Disclose Indonesia and Africa operations fully. Critical for national-security review.
    Disclose fully
  • ✓
    15. Potentially Duplicative Funding Notice
    PDF. Statement on any overlapping federal funding for same work. If none: simple “none” declaration.
    Simple declaration
  • ✓
    16. Environmental Questionnaire
    PDF. U.S. facility site location, potential environmental impacts, permitting status, NEPA information. Requires specific site data.
    Needs site data
  • ✓
    17–18. SF-LLL & Lobbying Certification
    PDF. Lobbying disclosures (if any) + standard certification that no federal funds used for lobbying.
    Short form
  • ✓

    Write last

3 Supporting Information (Embedded in Technical Volume & Evaluation)

Must be ready to provide — woven into Technical Volume or provided separately if requested.

  • ✓
    Feedstock Plan
    Volumes, sources (domestic e-waste + Indonesia + Africa), contracts or LOIs, traceability chain, diversification strategy.
  • ✓
    Offtake Agreements
    Buyer commitments with minimum annual tonnage. Use non-binding LOI template. Target: battery manufacturers, defense contractors, electronics firms.
  • ✓
    Technical Data & Process Details
    Recovery yields for each metal (silver/tungsten/copper/cobalt/zinc/lead). Process flow diagrams. Cost comparison vs. imported materials. Innovation claims.
  • ✓
    Job Creation & Community Benefits
    Number and types of jobs. Locations. Equity and community impact metrics. Workforce development plans.
  • ✓
    Intellectual Property & Data Management
    Any patents or proprietary processes. IP protection plan. Open-source commitments if applicable. Data management approach.
  • ✓
    Risk Assessment & Mitigation
    Supply risks, technical risks, financial risks, regulatory risks, market risks. Mitigation strategy for each. Contingency plans.
  • ✓
    Permitting & Site Status
    U.S. facility location confirmed. Zoning verification. Environmental permits in process or planned. Site control documentation.
